## 1. 機能概要

MedicalRecorder のメール送信機能では、録音した音声ファイルをメールに添付して送信できます。

### 主な機能

| 機能 | 説明 |
|-----|------|
| 録音ファイル一覧表示 | 保存されている録音ファイルを一覧で確認 |
| メール送信 | 選択したファイルをメールに添付して送信 |
| ファイル削除 | 不要なファイルを削除してストレージを管理 |

### 対応ファイル形式

- M4A（標準録音フォーマット）
- WAV
- MP3
- AAC

## 2. 事前準備

### 2.1 メールアカウントの設定

メール送信機能を使用するには、iPhoneにメールアカウントが設定されている必要があります。

**確認方法:**

1. iPhoneの「設定」アプリを開く
2. 「メール」をタップ
3. 「アカウント」をタップ
4. メールアカウントが登録されていることを確認

**アカウントがない場合:**

1. 「アカウントを追加」をタップ
2. メールプロバイダーを選択（iCloud、Gmail、Outlook等）
3. 画面の指示に従ってアカウントを追加

## 3. 基本操作

### 3.1 録音一覧画面を開く

1. MedicalRecorderアプリを起動
2. メイン画面下部の「**録音一覧**」ボタンをタップ

```
┌─────────────────────────────┐
│                             │
│      🎙️ 録音開始            │
│                             │
├─────────────────────────────┤
│ ┌──────────┐ ┌──────────┐   │
│ │インポート │ │録音一覧  │◄──│ ここをタップ
│ └──────────┘ └──────────┘   │
└─────────────────────────────┘
```

### 3.2 画面の見方

録音一覧画面では、以下の情報が表示されます：

```
┌─────────────────────────────────┐
│ recording_1703312345            │ ← ファイル名
│ 🕐 5:32  📄 2.1 MB              │ ← 録音時間 / ファイルサイズ
│ 2025年12月23日 14:30        📧  │ ← 作成日時 / 送信ボタン
└─────────────────────────────────┘
```

## 4. 録音ファイル一覧の操作

### 4.1 ファイル情報の確認

各ファイルには以下の情報が表示されます：

| 表示項目 | 説明 | 例 |
|---------|------|-----|
| ファイル名 | 録音時のタイムスタンプ | recording_1703312345 |
| 録音時間 | ファイルの再生時間 | 5:32 |
| ファイルサイズ | ファイルの容量 | 2.1 MB |
| 作成日時 | 録音した日時 | 2025年12月23日 14:30 |

### 4.2 ファイルの並び順

- ファイルは**作成日時の降順**（新しい順）で表示されます
- 最も新しい録音が一番上に表示されます

## 5. メール送信手順

### 5.1 送信したいファイルを選択

1. 録音一覧画面を開く
2. 送信したいファイルの行にある「**📧**」（封筒アイコン）をタップ

```
recording_1703312345
🕐 5:32  📄 2.1 MB
2025年12月23日 14:30        [📧] ← ここをタップ
```

### 5.2 メール作成画面

封筒アイコンをタップすると、メール作成画面が開きます。

**自動入力される内容:**

| 項目 | 内容 |
|-----|------|
| 件名 | 【MedicalRecorder】{ファイル名} |
| 本文 | MedicalRecorderから録音ファイルを送信します。<br>ファイル名: {ファイル名}.m4a |
| 添付ファイル | 選択した音声ファイル |

### 5.3 宛先の入力

1. 「宛先」フィールドをタップ
2. メールアドレスを入力、または連絡先から選択
3. 必要に応じてCc/Bccを追加

### 5.4 送信

1. 内容を確認
2. 右上の「**送信**」ボタンをタップ
3. 送信完了

**送信後の動作:**
- メール作成画面が閉じます
- 録音一覧画面に戻ります

## 6. ファイル削除

### 6.1 スワイプで削除

1. 削除したいファイルを左にスワイプ
2. 「**削除**」ボタンが表示される
3. 「削除」をタップして削除

```
┌─────────────────────────────────┐
│ recording_1703312345        │削除│
│ 🕐 5:32  📄 2.1 MB          │    │
└─────────────────────────────────┘
                              ↑
                          タップで削除
```

### 6.2 編集モードで削除

1. 右上の「**編集**」ボタンをタップ
2. 削除したいファイルの左側に表示される「⊖」をタップ
3. 「削除」をタップ
4. 終了したら「完了」をタップ

## 7. トラブルシューティング

### 7.1 「メールアカウントが設定されていません」と表示される

**原因:** iPhoneにメールアカウントが登録されていません。

**対処法:**
1. 「設定」アプリを開く
2. 「メール」→「アカウント」→「アカウントを追加」
3. メールアカウントを設定

### 7.2 メールが送信できない

**確認項目:**
- [ ] インターネットに接続されていますか？
- [ ] メールアカウントが正しく設定されていますか？
- [ ] 添付ファイルのサイズは制限内ですか？

**ファイルサイズ制限の目安:**
| メールサービス | 添付ファイル上限 |
|--------------|----------------|
| Gmail | 25 MB |
| iCloud | 20 MB |
| Outlook | 20 MB |
| Yahoo | 25 MB |

### 7.3 録音ファイルが表示されない

**確認項目:**
- [ ] 録音を行いましたか？
- [ ] 録音が正常に完了しましたか？

**対処法:**
1. アプリを完全に終了
2. アプリを再起動
3. 「録音一覧」を再度開く

### 7.4 ファイルが削除できない

**対処法:**
1. アプリを完全に終了
2. アプリを再起動
3. 再度削除を試行

## 8. よくある質問

### Q1. 複数のファイルを一度に送信できますか？

**A:** 現在のバージョンでは、1回の操作で1ファイルのみ送信できます。複数ファイルを送信する場合は、それぞれ個別に送信してください。

---

### Q2. 送信したメールはどこに保存されますか？

**A:** 送信済みメールは、お使いのメールアプリの「送信済み」フォルダに保存されます。

---

### Q3. 送信をキャンセルできますか？

**A:** メール作成画面で「キャンセル」をタップすると、送信せずに閉じることができます。下書きとして保存するか、破棄するかを選択できます。

---

### Q4. ファイルを削除しても復元できますか？

**A:** 一度削除したファイルは復元できません。重要なファイルは削除前にメール送信やバックアップを行ってください。

---

### Q5. メールの件名や本文を編集できますか？

**A:** はい。メール作成画面で件名・本文は自由に編集できます。

---

### Q6. 添付ファイルのサイズが大きすぎる場合は？

**A:** 以下の方法を検討してください：
- 短い録音に分けて送信
- AirDropやクラウドストレージを使用
- ファイル圧縮アプリを使用
