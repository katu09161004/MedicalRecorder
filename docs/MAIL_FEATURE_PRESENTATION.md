# MedicalRecorder メール送信機能
## 操作概要プレゼンテーション

---

## 📧 新機能の概要

### 録音ファイルをメールで簡単共有

- 録音した音声ファイルをワンタップでメール送信
- 標準のメールアプリを使用
- 件名・本文は自動入力

---

## 🎯 こんな時に便利

| シーン | 活用例 |
|-------|--------|
| 📋 会議後 | 議事録の音声を参加者に共有 |
| 🏥 診療後 | 診療メモを同僚に送信 |
| 📚 研修後 | 録音データをバックアップ |
| 💼 外出先 | PCにファイルを転送 |

---

## 🔧 操作の流れ

```
┌─────────────────┐
│  メイン画面     │

         ▼

│  録音ファイル   │
│  一覧表示       │◄── ② ファイル確認
└────────┬────────┘

└────────┬────────┘
         ▼
┌─────────────────┐
│  メール作成     │◄── ④ 宛先入力
│  画面           │
└────────┬────────┘
         ▼
┌─────────────────┐
│  送信完了！     │
└─────────────────┘
```

## ✉️ メール作成画面

### 自動入力される内容

| 項目 | 内容 |
|-----|------|
| 件名 | 【MedicalRecorder】recording_1703312345 |
| 本文 | MedicalRecorderから録音ファイルを送信します。<br>ファイル名: recording_1703312345.m4a |
| 添付 | 📎 recording_1703312345.m4a |

👆 **宛先だけ入力すればOK！**

---

## ⚠️ 注意事項

### メール設定が必要

```
❌ メールアカウント未設定の場合
   → アラートが表示されます
   → 「設定」アプリでメールを設定してください
```

### ファイルサイズに注意

```
📧 メールの添付制限
   ├── 一般的なメール: 10〜25MB
   ├── Gmail: 25MB
   └── iCloud: 20MB

💡 長時間録音は分割して送信を推奨
```

## ✅ まとめ

### 3ステップで簡単送信

```
① 「録音一覧」をタップ
        ▼
② 📧 アイコンをタップ
        ▼
③ 宛先を入力して送信
```

### ポイント

- 🔹 件名・本文は自動入力
- 🔹 ファイルは自動添付
- 🔹 標準メールアプリを使用
- 🔹 不要ファイルは簡単削除

---

## 📞 サポート

問題が発生した場合:

1. メールアカウントの設定を確認
2. ファイルサイズを確認
3. アプリを再起動
